# Queue System Project

מערכת לניהול תורים עם Node.js + MongoDB + React.

## 📦 התקנות ל־Backend

```bash
mkdir queue-system
cd queue-system
npm init -y
npm install express mongoose dotenv morgan
npm install --save-dev nodemon jest supertest
```

## 📦 התקנות ל־Frontend

npx create-react-app client
cd client
npm install
